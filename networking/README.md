## Agent-matrix Network Routing and configuration objecxts.
This directory contains the agent-matrix network routing and configuration objects.
### Homelab router gateway
The agent-matrix lab is gatewayed by a Linksys WRT32000ACM integrated, router, wifi hotspot, firewall, openvpn server and DNS server.
It is the homelab's gateway to the internet and runs the dd-wrt opensource router software, the DD-WRT director holds the actuasl config file as well as a human readable dump. For docuemtation and research purposes only
